# 이미지 텍스트 편집 평가 기준표

Planner, Generator, Evaluator가 공유하는 품질 기준.

---

## 채점 항목

### 1. 텍스트 정확도 (비중: 40%)
수정된 텍스트/숫자가 요청과 정확히 일치하는가?

합격: 모든 수정 대상 텍스트가 올바른 값으로 변경됨, 오타 없음, 누락 없음
불합격: 숫자 오류, 텍스트 누락, 잘못된 값, 일부만 수정됨

### 2. 시각 일관성 (비중: 30%)
수정된 텍스트의 폰트, 크기, 색상, 정렬이 원본과 일치하는가?

합격: 폰트 스타일 유사, 크기 동일, 색상 일치, 수평/수직 정렬 정확
불합격: 폰트가 눈에 띄게 다름, 크기 불일치, 색상 차이, 정렬 어긋남

### 3. 주변 보존 (비중: 20%)
수정하지 않은 영역이 원본 그대로 보존되었는가?

합격: 수정 대상 외 텍스트/이미지 손상 없음, 레이아웃 유지
불합격: 인접 텍스트 침범, 아이콘 손상, 배경 요소 훼손

### 4. 배경 자연스러움 (비중: 10%)
텍스트를 덮어쓴 영역의 배경이 자연스러운가?

합격: 원본 배경과 구분 불가, 그라데이션 연속성 유지
불합격: 박스 흔적 보임, 색상 단절, 인위적 느낌

---

## 판정

가중 점수 = (텍스트정확도×0.4) + (시각일관성×0.3) + (주변보존×0.2) + (배경자연스러움×0.1)

- 7.0 이상 → 합격
- 5.0~6.9 → 조건부 합격 (피드백 반영 후 재실행)
- 5.0 미만 → 불합격
- 텍스트 정확도 또는 주변 보존 4점 이하 → 무조건 불합격

---

## 피드백 형식

```
**전체 판정**: [합격 / 조건부 합격 / 불합격]
**가중 점수**: X.X / 10.0

**항목별 점수**:
- 텍스트 정확도: X/10 — [한 줄 코멘트]
- 시각 일관성: X/10 — [한 줄 코멘트]
- 주변 보존: X/10 — [한 줄 코멘트]
- 배경 자연스러움: X/10 — [한 줄 코멘트]

**구체적 개선 지시**:
1. [어디를 어떻게 고칠 것]
2. [어디를 어떻게 고칠 것]

**방향 판단**: [현재 방향 유지] 또는 [완전히 다른 접근 시도]
```
