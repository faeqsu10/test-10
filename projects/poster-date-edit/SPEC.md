## 편집 전략

### 배경 복원 방식: Column-wise interpolation (열 단위 보간)
- 배경이 수평 그라데이션이므로, 각 수정 영역에 대해 해당 열의 상하단 배경 픽셀을 샘플링하여 세로 방향으로 보간
- 구체적 절차:
  1. 수정 영역 상단 (y=302-305) 및 하단 (y=327-330) 배경 픽셀을 열(column)별로 샘플링
  2. 각 열에 대해 상-하 RGB 값을 선형 보간하여 배경 매트 생성
  3. 보간된 배경으로 원본 텍스트 영역 덮어쓰기
  4. 새 텍스트 렌더링
